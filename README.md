# PROJETO FPF — Dashboard de Performance

Dashboard que cruza performance de campanhas do Gerenciador de Anúncios da Meta
com leads reais de planilhas do Google Sheets, por Ação.

Contexto do projeto, convenções, design system e estado das fases: ver
[`CLAUDE.md`](./CLAUDE.md).

## Desenvolvimento

```bash
npm install
npm run dev
```

Abra [http://localhost:3000](http://localhost:3000).

```bash
npm run build   # build de produção
npm run lint    # eslint
```
